## 动物（被动型生物）

被动型生物是对玩家无害的可繁殖动物。

### 常见动物

| 动物 | 掉落物 | 繁殖食物 |
|------|-------|---------|
| 牛 | 生牛肉、皮革 | 小麦 |
| 羊 | 生羊肉、羊毛 | 小麦 |
| 猪 | 生猪肉 | 胡萝卜、马铃薯、甜菜根 |
| 鸡 | 生鸡肉、羽毛、鸡蛋 | 小麦种子 |
| 兔子 | 生兔肉、兔皮、兔子脚 | 蒲公英、胡萝卜 |
| 马 | 皮革 | 金苹果、金胡萝卜 |
| 驴/骡 | 皮革 | 金苹果、金胡萝卜 |
| 哞菇 | 生牛肉、蘑菇 | 小麦 |
| 狐狸 | 无 | 甜浆果（不会繁殖但可诱骗） |
| 狼 | 无 | 生肉（驯服后的狼会攻击目标） |
| 猫 | 线 | 生鱼（驯服后驱赶苦力怕） |
| 山羊 | 山羊角 | 小麦 |

### 动物繁殖

- 给两只同种动物喂食特定食物，它们会进入求爱模式并生下一只幼崽
- 幼崽需要 20 分钟成长（使用相应食物可加速成长）
- 动物会跟随手持食物的玩家
- 繁殖冷却：5 分钟

### 畜牧技巧

- 用栅栏围一块牧场，防止动物走散
- 大量种植小麦作为主要饲料
- 村民农民可自动收集和交易作物
- 分类饲养不同动物

### 驯服

- **狼**：手持骨头不断右键，出现爱心表示驯服
- **猫**：手持生鱼潜伏靠近，被信任后驯服
- **马/驴**：徒手右键骑乘直到不再踢人，之后可安装鞍
- **鹦鹉**：喂食小麦种子随机驯服
