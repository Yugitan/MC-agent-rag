## 村民

村民是 Minecraft 中的被动生物，自然生成在平原、沙漠、热带草原、针叶林和雪原村庄中。

### 村民职业

村民的职业由其工作站方块决定：

| 职业 | 工作站 | 交易物品 |
|------|--------|---------|
| 盔甲匠 | 高炉 | 铁/钻石盔甲，附魔钻石装备 |
| 屠夫 | 烟熏炉 | 生肉，熟肉 |
| 制图师 | 制图台 | 空地图，探险家地图 |
| 牧师 | 酿造台 | 红石粉，青金石，末影珍珠 |
| 农民 | 堆肥桶 | 食物，面包，蛋糕 |
| 渔夫 | 木桶 | 生鱼，熟鱼，附魔钓鱼竿 |
| 箭术师 | 制箭台 | 箭，附魔弓 |
| 皮匠 | 炼药锅 | 皮革，皮革盔甲 |
| 图书管理员 | 讲台 | 附魔书，书架，玻璃 |
| 石匠 | 切石机 | 石头方块，石英 |
| 牧羊人 | 织布机 | 羊毛，地毯，旗帜 |
| 工具匠 | 锻工台 | 工具，附魔工具 |
| 武器匠 | 砂轮 | 附魔剑，铁剑，钻石剑 |

### 交易机制

- **交易等级**：村民有新手、学徒、熟练工、专家、大师五个等级
- **经验提升**：与村民交易会获得经验，达到阈值后村民会升级并解锁新交易
- **打折机制**：治愈僵尸村民后，交易价格会大幅降低
- **繁殖**：村民上方有床并愿意时，给予面包（或胡萝卜/马铃薯/甜菜根）可使其繁殖

### 村民设施

- **村民繁殖机**：提供足够的床和食物让村民自动繁殖
- **交易大厅**：将不同职业村民集中管理，方便交易
- **铁傀儡农场**：村民被僵尸惊吓时会生成铁傀儡，可自动化获取铁锭
