## 树木

树木是 Minecraft 世界中最常见的资源，提供原木和树叶。

### 树的主要种类

| 种类 | 原木颜色 | 生成群系 |
|------|---------|---------|
| 橡树 | 棕色 | 森林、平原、沼泽 |
| 桦树 | 白色 | 桦木森林 |
| 云杉 | 深棕色 | 针叶林、雪原 |
| 丛林树 | 棕色 | 丛林 |
| 金合欢 | 灰色 | 热带草原 |
| 深色橡树 | 深棕色 | 黑森林 |
| 樱花树 | 粉红色原木 | 樱花树林 |

### 树木生长

- **树苗**：打掉树叶有小概率掉落树苗
- **骨粉**：对树苗使用骨粉可立即催生树木
- **光照要求**：树苗需要光照等级 >= 9
- **空间要求**：每种树的生长需要上方足够空间（橡树 5+ 格，云杉和丛林树需要更多）
- **大型变种**：云杉和丛林树有 2×2 大型变种

### 原木用途

- 合成木板（1 原木 → 4 木板）
- 烧制木炭（替代煤炭）
- 作为装饰性建筑材料
- 去皮后可用于建筑（用斧右键点击原木）

### 获取木材技巧

- 种植树苗实现可持续木材供应
- 深色橡树收益最高（1 个树苗长大可能产出 20+ 原木）
- 云杉 2×2 大型树效率极高
