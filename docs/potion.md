## 药水

药水可以给玩家和生物施加各种状态效果。药水酿造需要酿造台。

### 酿造基础

**酿造台**：1 个烈焰棒 + 3 个圆石合成

**粗制药水**：玻璃瓶装水 + 下界疣 → 粗制药水（所有药水的基础）

**二级材料**：
- 红石粉 → 延长持续时间
- 萤石粉 → 增强效果等级
- 火药 → 变为喷溅药水
- 龙息 → 变为滞留药水

### 基础药水配方

**粗制药水 + 以下材料：**

| 材料 | 效果 |
|------|------|
| 幽灵之泪 | 生命恢复药水 |
| 烈焰粉 | 力量药水 |
| 糖 | 迅捷药水 |
| 兔脚 | 跳跃提升药水 |
| 闪烁的西瓜片 | 瞬间治疗药水 |
| 发酵蛛眼 | 造成以下负面效果药水 |
| 金胡萝卜 | 夜视药水 |
| 河豚 | 水下呼吸药水 |
| 岩浆膏 | 抗火药水 |
| 蛛眼 | 剧毒药水 |
| 恶魂之泪 | 再生药水 |

### 负面药水

将正面药水用发酵蛛眼转化：
- 治疗药水 → 伤害药水
- 夜视药水 → 隐身药水
- 跳跃提升药水 → 缓慢药水
- 迅捷药水 → 迟缓药水
- 力量药水 → 虚弱药水

### 喷溅药水和滞留药水

- **喷溅药水**：基础药水 + 火药，可以投掷，对范围内的生物生效
- **滞留药水**：喷溅药水 + 龙息，投掷后留下药水云，经过的生物会受影响
