## 矿石

矿石是 Minecraft 中最重要的资源来源，开采后可以冶炼成金属锭或直接使用。

### 主世界矿石

| 矿石 | 冶炼产物 | 主要用途 |
|------|---------|---------|
| 煤矿石 | 煤炭 | 燃料、火把 |
| 铁矿石 | 铁锭 | 工具、盔甲、铁轨、漏斗等 |
| 铜矿石 | 铜锭 | 避雷针、望远镜、铜块 |
| 金矿石 | 金锭 | 动力铁轨、金苹果、下界合金 |
| 红石矿石 | 红石粉 | 红石电路、药水 |
| 青金石矿石 | 青金石 | 附魔、染料 |
| 钻石矿石 | 钻石 | 顶级工具和盔甲 |
| 绿宝石矿石 | 绿宝石 | 与村民交易 |
### 下界矿石

| 矿石 | 掉落/产物 | 主要用途 |
|------|----------|---------|
| 下界石英矿石 | 下界石英 | 红石比较器、探测铁轨、装饰 |
| 远古残骸 | 远古残骸碎片 | 下界合金锭 |
| 下界金矿石 | 金粒 | 金锭（9 粒合成 1 锭） |
### 矿石采集要点

- **精准采集附魔**：可以直接挖到矿石方块本身（适合装饰）
- **时运附魔**：增加矿石掉落数量（钻石、煤炭、青金石、红石、下界石英、铜等）
- **铁镐及以上**才能采集钻石、红石、金、绿宝石矿石
- **下界合金工具**：在熔岩中不会烧毁，高耐久
