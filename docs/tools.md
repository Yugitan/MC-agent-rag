## 工具

工具是 Minecraft 中提高采集效率和执行特定操作的物品。

### 镐（Pickaxe）

用于挖掘石头类方块、矿石等。采集矿石时必须使用正确材质的镐。

**合成配方**：3 个材料横排 + 2 根木棍竖排

**效率**：木镐 < 石镐 < 铁镐 < 金镐 < 钻石镐 < 下界合金镐

- **木镐**：可采石煤、圆石。耐久度低
- **石镐**：可采铁矿石、青金石矿石
- **铁镐**：可采钻石矿石、红石矿石、金矿石、绿宝石矿石
- **钻石镐**：可采黑曜石（需要 6.5 秒），古代残骸
- **下界合金镐**：最高耐久，不会在熔岩中烧毁

### 锹（Shovel）

用于挖掘泥土、沙子、沙砾、粘土等软质方块。

**合成配方**：1 个材料 + 2 根木棍竖排

### 斧（Axe）

用于砍伐原木，也可以作为近战武器使用（伤害高于剑但速度较慢）。

**合成配方**：3 个材料 + 2 根木棍（如镐但用 2 个材料而非 3 个，实际配方为对角线排列）

### 锄（Hoe）

用于将泥土、草方块转化为耕地，用于种植作物。

**合成配方**：2 个材料 + 2 根木棍（横排材料 + 竖排木棍）

### 钓鱼竿（Fishing Rod）

用于钓鱼。在有水的地方使用，等待浮标下沉后收杆。
可以钓到鱼、宝藏和垃圾。

### 剪刀（Shears）

用于采集树叶、羊毛，也可以修剪哞菇和蜂巢。

### 打火石（Flint and Steel）

用于点火，激活下界传送门。

### 指南针、时钟

- **指南针**：指向世界出生点
- **时钟**：显示当前游戏时间（白天/黑夜）
