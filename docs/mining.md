## 采矿

采矿是 Minecraft 的核心玩法之一，获取矿石和资源以制作更高级的装备。

### 矿石分布与层数

自 Minecraft 1.18 更新后，世界高度改为 -64 到 320 层。

**重要矿石的最佳层数（以 Y 坐标表示）：**

| 矿石 | 最集中层数 | 可用的最低级镐 |
|------|-----------|---------------|
| 煤矿石 | Y=96 和 Y=0 附近 | 木镐 |
| 铁矿石 | Y=16 和 Y=232 附近 | 石镐 |
| 铜矿石 | Y=48 附近 | 石镐 |
| 青金石矿石 | Y=0 附近 | 石镐 |
| 金矿石 | Y=-16 附近 | 铁镐 |
| 红石矿石 | Y=-32 附近 | 铁镐 |
| 钻石矿石 | Y=-64 至 -58（-59 最佳） | 铁镐 |
| 绿宝石矿石 | Y=236 附近 | 铁镐 |
| 古代残骸 | Y=15 附近（下界） | 钻石镐 |

### 采矿方法

**鱼骨挖矿法**：先挖一条主巷道，然后在两侧每隔 2-3 格挖分支巷道。效率高，覆盖面广。

**天然矿道探索**：探索已有的洞穴和峡谷，比人工挖掘更快找到矿石，但更危险。

**下井式挖矿法**：垂直挖到目标层数后，进行阶梯式或螺旋式挖掘。

### 挖矿安全

- 绝对不要直接向下挖（竖井挖法）而不站在两格交界处
- 随身携带水桶应对熔岩
- 带足火把防止刷怪
- 携带床在下界可用于设置重生点
