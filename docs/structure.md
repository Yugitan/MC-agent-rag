## 生成结构

生成结构是在世界中自然生成的独特建筑，包含宝箱和各种挑战。

### 主世界结构

| 结构 | 生成群系 | 特点 |
|------|---------|------|
| 村庄 | 平原、沙漠、针叶林、热带草原、雪原 | 村民+建筑，可交易 |
| 地牢 | 地下任意 | 苔石+圆石，内含刷怪箱和 1-2 个箱子 |
| 矿井 | 地下任意 | 木支架轨道，蜘蛛网，箱子矿车 |
| 要塞 | 地下 | 图书馆、传送门房间、末地传送门 |
| 沙漠神殿 | 沙漠 | TNT 陷阱+ 4 个箱子 |
| 丛林神庙 | 丛林 | 机关陷阱+红石+箱子 |
| 沼泽小屋 | 沼泽 | 女巫住的小屋，内有炼药锅 |
| 雪屋 | 雪原 | 可能有地下室，地下室有村民和僵尸村民 |
| 海底遗迹 | 深海 | 守卫者、远古守卫者，内含海绵和金块 |
| 海底废墟 | 海洋/海滩 | 水下残骸，箱子 |
| 林地府邸 | 黑森林 | 唤魔者、卫道士，战利品丰富 |
| 掠夺者前哨站 | 任意陆地 | 掠夺者塔，周围有掠夺者 |

### 下界结构

| 结构 | 特点 |
|------|------|
| 下界要塞 | 烈焰人刷怪箱、下界疣、凋零骷髅生成 |
| 堡垒遗迹 | 猪灵堡垒，金块和下界合金装备 |

### 末地结构

| 结构 | 特点 |
|------|------|
| 末地城 | 紫珀块建筑，内部箱子有钻石装备 |
| 末地船 | 悬浮在末地城旁边，获得鞘翅和龙头 |
