## 酿造

酿造是使用酿造台制作药水的工艺。

### 酿造台

**合成**：1 个烈焰棒 + 3 个圆石

放置后右键打开，下方放烈焰粉作为燃料（每个可酿 20 次）。

### 酿造步骤

**第一步：制作粗制药水**
玻璃瓶装水 + 下界疣（放在上方材料槽）→ 粗制药水

**第二步：添加基础效果材料**
粗制药水 + 对应材料 → 基础效果药水

**第三步：增强药水**
- 红石粉 → 延长持续时间（8:00）
- 萤石粉 → 提升效果等级（如 II 级）
- 红石和萤石不能同时使用

**第四步：转化**
- 火药 → 喷溅药水（可投掷）
- 龙息 → 滞留药水（留下药水云）

### 药水配方汇总

| 药水 | 材料 | 效果 |
|------|------|------|
| 粗制药水 | 下界疣 | 基础（无效果） |
| 治疗药水 | 闪烁的西瓜片 | 瞬间恢复 4 ❤ |
| 伤害药水 | 发酵蛛眼 + 治疗药水 | 瞬间伤害 |
| 再生药水 | 恶魂之泪 | 每 2.5 秒恢复 1 ❤，0:45 |
| 力量药水 | 烈焰粉 | 攻击力 +3，3:00 |
| 迅捷药水 | 糖 | 速度 +20%，3:00 |
| 迟缓药水 | 发酵蛛眼 + 迅捷药水 | 速度 -15%，1:30 |
| 夜视药水 | 金胡萝卜 | 夜视，3:00 |
| 隐身药水 | 发酵蛛眼 + 夜视药水 | 隐身，3:00 |
| 抗火药水 | 岩浆膏 | 免疫火/熔岩，3:00 |
| 水下呼吸药水 | 河豚 | 水下呼吸，3:00 |
| 跳跃药水 | 兔脚 | 跳跃提升，3:00 |
| 虚弱药水 | 发酵蛛眼 | 攻击力 -4，1:30 |

### 药水增强

红石粉延长持续时间（效果变 8:00），萤石粉增强效果等级（如再生 II、力量 II）。
注意：萤石粉和红石粉互斥，只能选其一。
